What changed from the previous mock

Suggestive instead of aggressive

  • No gradient hero. Heading is "Optional prep for your technical screen"
  • Three peer cards instead of one giant CTA — candidate scans, picks, or ignores
  • Each card has a "💡 Helpful when..." note explaining the right moment to use it
  • Friendly footer panel reassures: "we're not grading you here"

Meta-style polish

  • White background, generous padding (36px page padding, 20px in cards)
  • Meta-blue (#1877f2) for accents, neutral greys (#65676b, #8a8d91) for body
  • Pill-shaped "View" button matches Meta exactly
  • Timeline dots + line use the same structure as Meta's app card
  • Light icon chips (#e7f3ff blue tint) instead of bold gradient blocks
